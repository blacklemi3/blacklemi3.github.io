# 公开项目记录页

这个目录是静态项目入口页，可直接部署到 GitHub Pages。

线上地址：

- https://blacklemi3.github.io/

入口文件：

- `index.html`
- `styles.css`
- `assets/profile.jpg`
- `cases/system-records.html`
- `cases/delivery-record.html`

页面当前采用“入口清晰、案例优先”的结构：

- 顶部导航直接进入真实系统记录、公开仓库和交付记录。
- 首页先给推荐阅读顺序，再给五条真实项目线的直达入口。
- `cases/system-records.html` 先给项目地图，再按硬件与设备、应用产品、工程交付展开。

真实系统记录：

- 双 MCU 机器人控制链路
- 嵌入式外设驱动与试验箱案例
- “喝啥”消费级轻产品 App
- 网络变更与自动回滚
- 边缘视觉训练与端侧部署

页面当前链接到 7 个公开仓库：

- `local-handover-workbench`
- `local-project-indexer`
- `admission-data-workbench`
- `static-site-delivery-kit`
- `edge-ai-vision-deployment-kit`
- `wechat-article-for-ai`
- `esp32-c3-ble-control-demos`

本页不依赖构建工具。
